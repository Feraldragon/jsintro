Prompting to create a list

Instructions

Attach onclick handler for the "Ask" button that does the following:
  1. Prompt the user for input
  2. If input is received, add value to stack array
Attach onclick handler to the "Tell" button that does the following:
  1. Joins the values in the stack array into a string
  2. The joined values should be a comma-separated string
  3. Display an alert using the joined string
Too easy? Try these additional steps:
  1. Create buttons for pop() and unshift()
  2. Create a button that sorts the array (and alerts with sorted values)
  3. Create a button to clear the list
  4. Tell button should alert with "No items." if array is empty.
