Generate a Deck of Cards, Shuffle, and Deal

Instructions

  • Two arrays are provided, suits and cards.
  • An example loop is also provided, that loops over suits.
  1. Write another for loop (inside the first one) to loop over cards.
  2. Inside the loops, generate a string for the name of the card (e.g. "Two of Hearts")
  3. Add the string to the deck using the Array.push() method.
  4. When the loops finish, use the _.shuffle(array) method from the underscore library, to shuffle the deck.
  5. Finally, write a function that deals a card from the deck using Array.shift() and displays its name in an alert() when the button is clicked.
